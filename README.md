## git-browser project
Allows to browse github's public repos, using github API

## Quick start
To install dependencies and clean the git repo run:</br>
  $ yarn install </br>
  or</br>
  $ npm install</br>


To run in dev mode:</br>
  $ yarn run dev</br>
    or</br>
  $ npm run dev</br>

[Test it on Heroku](https://g-git-browser.herokuapp.com/)</br>
